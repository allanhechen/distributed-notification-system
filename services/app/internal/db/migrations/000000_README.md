We use golang-migrate to handle our database migrations.

To migrate up one transaction:

```bash
cd ./services/app
export DATABASE_URL="your local database url for CockroachDB"
migrate -database ${DATABASE_URL} -path db/migrations up 1
```

To migrate down one transaction:

```bash
cd ./services/app
export DATABASE_URL="your local database url for CockroachDB"
migrate -database ${DATABASE_URL} -path db/migrations down 1
```

All transactions should be idempotent. Running the same migration multiple
times should not cause any issues.
